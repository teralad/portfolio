# Portfolio

My portfolio ever since I started my journey as a developer.

## Accessing the Application

### Live Site

The portfolio is a static website. Open `index.html` directly in any modern browser:

```
open index.html
```

Or serve it locally with any static file server. For example, using Python:

```bash
python3 -m http.server 8000
```

Then visit [http://localhost:8000](http://localhost:8000) in your browser.

Alternatively, with Node.js (`npx`):

```bash
npx serve .
```

Then visit the URL shown in the terminal (usually [http://localhost:3000](http://localhost:3000)).

### Sections

The site includes the following sections, accessible via the top navigation bar:

- **About** — Background and tech stack
- **Projects** — Featured projects (Gifle, Stack Wizard, Hashira)
- **Skills** — Languages, frontend, backend, DevOps, and databases
- **Contact** — Email, GitHub, LinkedIn, and X (Twitter)
